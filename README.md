# Cabinet Ines

Site vitrine React/Vite pour le cabinet de diététique fonctionnelle Ines.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Structure

- `src/components` : composants de layout réutilisables.
- `src/pages` : pages publiques du site.
- `src/data/siteData.js` : contenus, liens et coordonnées centralisés.
- `src/assets` : images et feuille de style globale.

Le build de production est généré dans `dist`.
